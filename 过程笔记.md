

## 个人博客 app


### 数据库配置过程
要开始使用，请先配置您的 Neon 数据库连接。

1. 连接数据库， vercel + neon 

2. 更新 .env 文件中的 DATABASE_URL

3. 运行 npm run db:push 初始化数据库

4. 运行 npm run db:seed 添加示例数据

详细说明请查看 SETUP.md 文件。
